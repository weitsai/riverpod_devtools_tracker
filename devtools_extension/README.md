# Riverpod DevTools Extension

This is the DevTools extension component of the `riverpod_devtools_tracker` package.

**Note**: This package is not meant to be used directly. It's automatically included when you use the main `riverpod_devtools_tracker` package.

## Main Package

For usage instructions and documentation, please see:
- **Main Package**: [riverpod_devtools_tracker](https://pub.dev/packages/riverpod_devtools_tracker)
- **GitHub**: [github.com/weitsai/riverpod_devtools_tracker](https://github.com/weitsai/riverpod_devtools_tracker)

## Features

This DevTools extension provides:
- Real-time monitoring of Riverpod state changes
- Interactive provider list with filtering
- Detailed call chain visualization
- GitHub-style dark theme UI
- Multi-language support (English, 繁體中文)

## Installation

The extension is automatically discovered by Flutter DevTools when you install the main package:

```yaml
dependencies:
  riverpod_devtools_tracker: ^1.0.0
```

No additional configuration needed!
